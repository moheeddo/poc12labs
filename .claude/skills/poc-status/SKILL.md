# POC 프로젝트 진행 상황 확인

사용자가 `/poc-status`를 호출하면 아래를 수행합니다.

## 점검 항목

### 1. Phase 진행도 확인
CLAUDE.md의 개발 순서(Phase 1~6)를 기준으로:
- 프로젝트 디렉토리 구조를 스캔하여 현재 어떤 Phase까지 완료되었는지 판단
- 각 Phase의 완료/진행중/미시작 상태 표시

### 2. 서비스별 구현 상태
3개 서비스 각각에 대해:
- **N-HPAI 시뮬레이터 평가**: 영상 업로드, 검색, 챕터링, 평가 리포트
- **리더십코칭 역량진단**: 발표자 분리, 스코어링, 피드백 리포트, 성장 추이
- **POV 분석**: SOP 이탈 탐지, 유사도 비교, 하이라이트 추출, 비교 뷰

### 3. TwelveLabs 연동 상태
- MCP 도구로 `list-indexes` 호출하여 인덱스 존재 여부 확인
- API 라우트 파일 존재 여부 확인
- 업로드된 영상 목록 확인

```

🔜 다음 단계 제안: ...
```
