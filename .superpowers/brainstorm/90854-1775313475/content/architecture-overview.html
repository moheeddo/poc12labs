PHASE 3 — AI 분석 파이프라인 (핵심)
3A. 단계 검출

→ 수행/미수행 판정
3B. 손-물체 분석
Pegasus Analyze
"손에 들고 있는 도구는?"
"밸브 조작 전후 상태는?"
→ 물체 ID + 상태변화 추출
3C. 시퀀스 매칭
DTW Algorithm
검출된 단계 순서 vs SOP 순서
→ 누락/순서오류/추가행동
→ 이탈 구간 타임라인
3D. HPO 도구 적용 검증
Marengo Search + Pegasus
11개 HPO 도구 적용 여부 검색
"자기진단(STAR) 수행 장면"
"동료점검 수행 장면"
3E. 임베딩 비교
Marengo Embed
골드스탠다드 vs 평가 영상
→ 세그먼트별 코사인 유사도
→ 정량적 격차 히트맵
3F. 종합 스코어링
Scoring Engine
절차준수(40%) + HPO(30%)
+ 기본수칙(20%) + 유사도(10%)
→ S/A/B/C/D/F 등급
